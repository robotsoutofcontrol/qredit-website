

# Qredit Website



```
git clone https://github.com/Qredit/qredit-website
cd qredit-website
npm install
node qredit.js
```

for live editing use "pm2 start qredit.js --watch" instead of "qredit app.js".

```
pm2 start qredit.js --watch
```
Website visible at "https://localhost:8080"
